Installing ADE Build Tools¶

ADE base tools¶

This installs the essential ADE build tools and scripts.

mkdir -p /gem_base
cd /gem_base
svn co http://sbfsvn02/gemini-sw/gem/vendor/support/Gemini-ADE/current ade
cd buildGemApp
make
